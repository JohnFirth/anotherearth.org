welcome to anotherearth.org

This application allows you to compare two maps easily and comprehensively, using the Google Earth browser plugin. An example use would be comparing two maps of the same place at different times; noteworthy comparisons can be saved as URL links. Movement of the Earths can be synchronized, and features such as buildings, roads and borders added.

Please refer to Google's documentation for guides to Google Earth and its navigation control.

If your browser's preferred language isn't English, and you have JavaScript enabled, then using a translator, powered by Google, I've attempted to convert the text.

Feel free contact me by email: contact(at)anotherearth(dot)org

I'm sorry, but this application is no longer compatible with Internet Explorer 6. If you are using this, I recommend using either one of the Mozilla Firefox or Google Chrome web browsers instead.
